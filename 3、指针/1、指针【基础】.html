指针
  • 都包含一个地址信息和一个类型（长度）信息。
  • 只能指向相同数据类型的变量/指针
    【地址按字节编址，计算机要知道储存对象需要多少字节。否则指针无法正确地取回地址上的值】
  • 对指针进行的加减，就是对指针的偏移。【加——向后；减——向前】。其值会增加/减少对应类型大小的数值【指向下一个存储单元】
  • 禁止对还没有指定地址的指针变量进行解引用。
  • 当有两个以上的运算符，建议使用圆括号。
type *p = 值;              // 【禁止】因为指针还没有指向某个地址。




指针变量使用
type *p1 = 初始地址值;       // 地址值的表现形式 &另一变量
*p1 = 值;                   // 通过指针变量修改值 

type *p2;
p2 = 初始地址值;   
*p2 = 值;
var1 = ++(*p);            // 1对值进行+1；2取值。
var1 = (*p)++;
var2 = *(++p);            // 1指针向后偏移；2取值。
var3 = *(p++);            // 1取值；2指针向后偏移。
